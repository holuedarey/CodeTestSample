# wiustore

### Simple Product Listing App.

This project is built using PHP as backend and Angular in the frontend.

### Frontend Operations
- List products
- List comments
- Add comment
- Infinite scroll
- Back breadcrumbs to return back to main page.

### API Options
- Product CRUD with pagination
- COmment CRUD with pagination
